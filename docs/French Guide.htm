SimpleDivX possède 9 onglets situés en haut de la fenêtre du programme, chacun étant désigné par un numéro et une description. Ce guide vous emmènera dans le programme onglet par onglet.
Pour plus de confort et de compréhension, il est possible de mettre SimpleDivX en français (voir l'étape 9).


Au premier démarrage, SimpleDivX devrais ressembler à l'image ci-dessous:


1ère étape: Input

Folder (Dossier):

Cliquez sur le petit bouton (...) situé à droite et une boîte telle que celle représentée ci-dessous devrait apparaître.

Sélectionnez le dossier dans lequel vous avez rippé vos fichiers VOB en utilisant Smartripper ou un autre utilitaire de rip et cliquez ensuite sur OK.

Vob Files (fichiers vob):

Si vous avez utilisé l'onglet "files" avec smartripper, vous pouvez cocher "select all files", par contre, si vous avez utilisé l'onglet "movie" ou un autre utilitaire de rip, vous devriez cocher "select main movie". Il est également possible de sélectionner les VOB un par un.

Video Information (informations sur la vidéo):

Cette partie vous donnera les principales informations sur le DVD que vous êtes entrain de sauvegarder. Très cool en effet :)


2ème étape: Video

Frames/Second (fps) (Images / Seconde):

Vous sélectionnez ici le taux d'images par seconde de la "sortie". Les utilisateurs PAL sélectionneront 25.000, tandis que les NTSC préféreront 23.976 ou 29.970. SimpleDivX sélectionne normalement le taux d'images par seconde détecté dans les fichiers vob.

Enable Video Stream ID checking (Active la recherche du flux video):

Cette option détectera automatiquement l'ID du flux vidéo principal. (Normalement 0xE0)

Auto-Retrieve Crop And Resizing Info (Coupe et Redimension Automatique):

Cochez la case : "Auto-Retrieve cropping resizing info" pour trouver automatiquement la taille correcte de l'image et sélectionner automatiquement les valeures de coupe et de redimension.

Output Format (Format de Sortie):

Sélectionnez le format de sortie que vous désirez. Les formats les plus employés sont prédéfinis, mais vous pouvez sélectionner 'custom' pour entrer votre propre format.

Open VOB Bitstream at LBA (Ouvrir le flux VOB au LBA):

(Utilisateurs expérimentés): Cochez cette case si vous voulez commencer le film DVD à partir d'une LBA spécifique.

Frames Range (Tranche d'Images):

Ici vous pouvez entrer le nombre exact d'image que vous désirez encoder. Cette option est très utile afin d'encoder de petites parties de films et les visualiser ensuite dans un lecteur pour en déterminer la qualité.

Advanced Video Settings (Propriétés avancées de la vidéo)


Lorsque vous appuyez sur le bouton 'advanced', le sous menu ci-dessus apparaît :

Postfilter Quality (Filtre de qualité):

- High, meilleure qualité / encodage plus long
- Low, moins bonne qualité / encodage plus rapide
(Laissez le sur 'High' pour de meilleurs résultats.)

Reference Quality:

Les différences entre les différentes options sont assez difficiles à expliquer. Pour faire simple, IDCT est utilisé pour les "Discrete Cosine Transform".
Nous vous suggérons d'utiliser ces quelques sélections en fonction de votre processeur:
Pentium MMX: 16-bit MMX ichen
PentiumII: 32-bit MMX
PentiumIII: 32-bit MMX
PentiumIV: 64-bit FPU
K6-2/3: 16-bit MMX ichen
Athlon/Duron: 32-bit MMX
Athlon XP: 32-bit MMX

Output Mode (Mode de Sortie):

SimpleDivX peut utiliser toute une série de formats de sortie, cependant nous vous recommandons yv12 étant donné que le DivX est actuellement le seul codec supporté par SimpleDivX.


3ème étape: Audio

Create Audio (Créer l'Audio):

Désélectionnez cette case si vous ne souhaitez pas que SimpleDivX crée l'audio dans votre fichier vidéo.

Enable Audio Stream ID checking (Activer la possibilité de sélectionner l'ID du flux audio):

Lorsque cette option est activée, vous n'avez qu'à sélectionner le flux audio que vous désirez joindre à la vidéo (Il est recommandé de laisser cette option activée)

Extract all AC3 Audio Streams (Extraire tous les flux Audio AC3):

Cela extraira tous les flux audio présents dans le film.
*NOTE* Seul le flux Audio sélectionné sera multiplexé avec la vidéo.

Audio Stream ID (ID du Flux Audio):

Cela liste tous les flux audio présents dans le film. Le flux audio principal sera pratiquement toujours 0x80, tandis que les autres seront la plupart du temps les autres langues, commentaires, ou bandes audio spéciales.

Audio Language (Language de l'Audio):

Lorsque les fichiers *.ifo du DVD sont aussi copiés, SimpleDivX est capable de vous préciser la langue contenue dans le flux audio sélectionné.

Audio kbps (Débit du flux audio):

Utile afin de décider si l'on compressera l'audio.

Audio Format (Format Audio):

MP3: Compresse le flux audio au format MP3 (Stéréo uniquement)
AC3: Conserve la qualité audio d'origine (Il peut s'agir de canaux Dobly-Digital Stereo, 5.0 ou 5.1)
OGG: Compresse le flux audio au format OGG (Stéréo uniquement)

Audio Compression kbps (Débit du Flux Audio Compressé):

Choisissez ici le taux de compression voulu pour le format audio.
Pour le MP3, 128/162Kbps est recommandé.
Si vous choisissez la compression en OGG, la qualité du son est bien meilleure à un taux plus bas, donc pour les films à 1CD, vous pouvez choisir 64Kbps. 112/160 sont normalement utilisés.
Lorsque AC3 est utilisé comme format audio, il est possible de recompresser ce flux à un taux plus bas. Mais pour les films à 6 canaux, il est recommandé de choisir au minimum 256Kbps.

Samplerate Conversion (Débit de conversion):

Pour le MP3 et l'OGG, il est possible de choisir comme débit soit 48000Hz, soit 44100Hz. Si votre carte son supporte 48000Hz, laissez 48000.

Amplify (Amplification):

Sélectionnez ceci pour amplifier le son de quelque dB. Il est recommandé de ne pas choisir une valeur plus grande que 3dB, ou vous entendrez des craquement dans les parties ou le son est plus fort.

Peak level normalize (Normalisation des Niveaux de Piques)

Lorsque cette option est sélectionnée, le son sera normalisé à 100% (Ce qui veux dire que le son le plus fort sera considéré comme 100%, donc aucune coupure)

Peak level normalize [midnight mode] (Normalisation des Niveaux de Piques [mode midnight])

Lorsque cette option est sélectionnée, le son sera normalisé à 100% et le son le plus doux sera également amplifié.


4ème étape: Codec


 

1-Pass DivX file (DivX à 1 passage):

Les DivX à 1 passage sont créé en utilisant un débit constant, ce qui signifie que toutes les images du film auront exactement le même débit.

2-Pass DivX file (DivX à 2 passages):

Les DivX à 2 passages sont créés en utilisant un débit variable, ce qui signifie que les scènes d'action bénéficieront d'un débit plus élevé que les scènes "calmes". Cela débouche sur un film de meilleur qualité qu'un DivX à 1 passage mais l'encodage dure à peu près deux fois plus longtemps.

Do Only First Pass (Ne faire que le premier passage):

SimpleDivX ne créera que le premier passage du film, lorsque ce sera fini vous devrez revenir dans cet onglet et sélectionner "Do Only Second Pass". Ceci est utile lorsque vous n'avez accès à un ordinateur que pour un temps limité.

Advanced Settings (Options Avancées)

Lorsque vous sélectionnez le codec "DivX 5.x", les options avancées suivantes vont apparaitre :


Lorsque "Xvid" est sélectionné, l'écran suivant va apparaître :

Les options suivantes sont configurable :

Global Motion Compensation (Compensation globale du mouvement):

GMC aide dans les opérations de zoom et de panorama.

Quarter Pixel (Quarts de Pixels):

Quarter pixel augmente la précision des macro blocs de votre film.

Bidirectional Encoding (Encodage bidirectionnel):

Peut aider à augmenter la compression dans certaines scènes en utilisant une prédiction en avant et en arrière des images.

De-Interlace (Désentrelacer)

Vous pouvez utiliser la fonction de désentrelaçage depuis le DivX 5.x PRO / XviD.

Get Time From Movie (Récupérer la durée depuis le film):

Cette option essayera de lire la durée du film depuis les fichiers *.vob. Ce n'est pas toujours précis et donc, si la durée s'avère fausse, entrez manuellement celle qui est spécifiée à l'arrière de la boîte de votre DVD.

End Credits Options (Générique de fin)

SimpleDivX proposes trois traitements différents pour le générique de fin.
- Vous pouvez le compressez normalement (utilise le même débit que le film)
- Ne pas les inclure
- Les compresser avec un débit plus faible (et donc "garder du débit" (ou garder de la bande passante) pour le film)

 

Tab 6: Misc


Pour le moment, seuls les sous-titres sont inclus sur cette page. Plus d'options seront ajoutées plus tard.

Create subtitles (Créer des sous-titres)

Cochez ceci pour activer le traitement des sous-titres.

Language

Sélectionnez ici le ou les langues des sous-titres qui devront être inclus.

Select all languages (Séléction de toutes les langues)

Sélectionner cette option reviendra à cocher touts les sous-titres présents sur le DVD. Cliquez une fois de plus pour désélectionner toutes les langues.

*Note*
Les sous-titres seront extraits en utilisant vobsub.dll, cette dll crée deux fichiers supplémentaires dans le dossier final. Pour voir les sous-titres, vous devez répondre aux critères suivants:
- Assurez vous que les deux fichiers soient dans le même dossier que le fichier final et qu'ils aient le même nom. (SimpleDivX devrait les renommer pour vous.)
1) Télécharger le pack entier de vobsub à l'adresse suivante : http://www.divx-digest.com/software/vobsub.html, maintenant vous devriez être capable de voir les sous-titres en utilisant n'importe quel lecteur vidéo dans Windows.
2) Téléchargez et installez le lecteur "Media Player Classic" (de préférence) sur : http://gabest.org/ ou http://sourceforge.net/projects/guliverkli. Ce lecteur est du même auteur que VobSub et inclus automatiquement un support pour les sous-titres.

 

Tab 7: Project

Project Name (nom du projet):

Vous pouvez entrer n'importe quel nom de projet, la plupart des personnes utilisent juste le nom du film. Au contraire de certains programmes utilisant pour vos fichiers des noms tels que "movie1.avi", etc., SimpleDivx utilisera pour la plupart des fichiers générés ce nom de projet.

Temp Folder (Dossier Temp):

C'est là que les fichiers vidéo, sonores et log temporaires seront sauvés.

Output Folder (Dossier de sortie):

C'est la que votre fichier final sera transféré. (Ce peut être un fichier *.avi ou un *.ogm)

Overwrite files (Réécrire le fichier):

Sélectionnez ceci pour réécrire automatiquement le fichier de destination (si celui-ci existe déjà).

Delete temp-files (Effacer les fichiers temporaires):

Sélectionnez ceci pour effacer automatiquement les fichiers générés par SimpleDivX se trouvant dans le dossier Temp.

Split Total File (Couper le fichier entier):

Si vous faites un Rip de 2CDs, SimpleDivX peut couper le fichier final automatiquement grâce à cette option.

Normal:

Dans ce mode, il n'est possible de ne faire qu'une conversion à la fois. (C'est le mode de compatibilité avec SimpleDivX 1.1)

Batch Mode (Mode séquentiel):

Cette option permet de programmer plusieurs encodages de plusieurs films en une seule session de SimpleDivX.

 

8ème étape: Output

Shutdown windows (Fermeture de Windows):

Cette option activée permettra l'extinction de l'ordinateur à la fin de l'encodage. Appuyez sur le bouton 'Start' pour commencer le processus.

Status Information (Etat):

Cette fenêtre vous tiendra informé de la progression de SimpleDivX.


9ème étape: Setup

External Program Locations (Chemins des programmes externes):

Cliquez sur le petit bouton (...) à droite et la boîte çi-dessous devrait apparaître.

Vous pouvez maintenant changer les trois chemins afin de les faire pointer vers les dossiers contenant les programmes nécessaires. Cette option est conçue pour les utilisateurs qui pourraient avoir déjà installé ces programmes sur leur ordinateur, cependant, étant donné que SimpleDivX inclus toujours les dernières versions de ces programmes, nous vous conseillons de laisser ces chemins pointer vers le répertoire d'installation de SimpleDivX.

Advanced Audio Settings (Options audio avancées)

Si l'assemblage avi / ac3 pose problèmes, vous pouvez changer les valeurs de l'interleave (interfoliage, assemblage) et du preload (préchargement).

Priority Settings (Priorité)

Choisissez ici les priorités pour :
- L'Audio
- Le Vidéo
- Les opérations de lecture / écriture sur les disques
 (Note : Placer les priorités sur des valeurs plus hautes [tel que "high"] ne veut pas dire que la compression ira toujours plus vite! Cela signifie juste que SimpleDivX aura droit à plus de cycles processeurs, autrement dit, il aura priorité sur le processeur par rapport aux autres programmes... ce qui provoquera un ralentissement de ces derniers.)

Language

Sélectionnez ici la langue dans laquelle vous voulez que SimpleDivX soit.
